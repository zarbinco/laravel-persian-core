# Support

For questions, bug reports, and feature discussions, use GitHub issues:

https://github.com/zarbinco/laravel-persian-core/issues

Before opening an issue, please include:

- Package version.
- PHP and Laravel versions.
- Relevant config values.
- A small reproduction case.
- Expected and actual behavior.

For security reports, use the process in [SECURITY.md](SECURITY.md).
